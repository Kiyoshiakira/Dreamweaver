## ğŸ¯ What is Dreamweaver?

Dreamweaver is a single-page web application that transforms your story ideas into immersive audiovisual experiences. Using advanced AI technologies, it:

1. **Generates** original stories based on your prompts
2. **Narrates** them with professional voice synthesis
3. **Visualizes** scenes with AI-generated images
4. **Scores** the narrative with intelligently-selected music

Perfect for:
- ğŸ˜´ **Bedtime Stories** - Relax as AI weaves tales to help you sleep
- ğŸ¨ **Creative Inspiration** - Explore story ideas and narrative possibilities

## ğŸ’¡ Quick Start

1. **Get API Keys**: Obtain a Google Gemini API key
2. **Configure**: Add your API key to `dreamweaver.html`
3. **Run**: Open the HTML file in a modern web browser
4. **Create**: Enter a story prompt and click "Begin Story"
5. **Enjoy**: Sit back and experience your personalized narrative
